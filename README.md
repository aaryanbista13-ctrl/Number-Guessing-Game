# Number Guessing Game

A Python command line game where the computer picks a random number 
and the player tries to guess it with helpful hints.

# How it works
- Computer picks a random number between 8 and 66
- Player guesses the number
- Program tells you if your guess is too high or too low
- Counts how many guesses it took to win

# How to run
python practice2.py
